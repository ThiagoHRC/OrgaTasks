# OrgaTasks
Aplicativo acadêmico de gerenciamento de tarefas. Inspirado no Trello para organização de tarefas. Desennvolvido em grupo para projeto de faculdade.

## Tecnologias
- **Back-end** Node.js, Express, PostgreSQL (Prisma ORM)
- **Front-end** React Native (Expo)
- **Controle de Versão** GitFlow

## MVP
- Cadastro de usuário
- CRUD de quadros, listas e cartões
- Mover cartões entre listas

## Setup
1. Clone o repo: `git clone https://github.com/ThiagoHRC/OrgaTasks.git`
2. Back-end: `cd backend && npm install`
3. Front-end: `cd frontend && npx expo start`
